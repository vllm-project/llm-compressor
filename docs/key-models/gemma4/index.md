# Gemma 4

Quantization examples for Google's Gemma 4 models.

> **Note:** These examples require `transformers >= v5`, which can be installed with:
> ```bash
> uv pip install --upgrade transformers==5.8.1
> ```

## Examples

- [NVFP4 Example](nvfp4-example.md)
- [FP8 Block Example](fp8-block-example.md)
- [NVFP4 MoE Example](nvfp4-moe-example.md)
