# DeepSeek V3.2

Quantization examples for DeepSeek's V3.2 model with Multi-Latent Attention (MLA) and disk offloading support.

> **Note:** DeepSeek V3.2 requires a custom model definition included in LLM Compressor, as it is not available in transformers. The model definition supports bfloat16 weights and data parallelism for compatibility with `oneshot`.

- [NVFP4-FP8-BLOCK Example](nvfp4-fp8-example.md)
